# AI Chatbot using Python

This is a beginner AI chatbot built using Python and basic Natural Language Processing.

## Features
- Responds to user greetings
- Answers basic questions
- Simple intent recognition

## Technologies Used
- Python
- NLTK
- NumPy

## Goal
This project was built while learning AI/ML fundamentals and Natural Language Processing.

- Add better NLP models
